# Playwright Automation Suite

A collection of end-to-end test automation scripts built with Playwright and TypeScript.

## Tech Stack
- Playwright
- TypeScript
- Page Object Model (POM)
- Node.js

## Folder Structure
- `tests/` - Test spec files
- `pages/` - POM classes
- `utils/` - Utilities and helpers

## How to Run
npm install
npx playwright test
